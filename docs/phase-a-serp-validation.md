## Required Live Check Before Drafting

For each page, capture:

- Top 5 ranking URLs for the primary query.
- Competitor type for each ranking URL.
- Whether AI/overview answers satisfy the query without a click.
- Whether the page can add a real gap: conservative TSA guidance, no-knife setup, pocket burden, tested/handled/researched labels, direct affiliate fit, or better beginner framing.
- Final decision: `build`, `reframe`, `defer`, or `drop`.

## Page Six Rule

Build `/best-keychain-edc-without-bulk/` only if the live SERP shows a gap for beginner-safe, low-bulk, non-novelty keychain carry. Use `/best-edc-kit-under-50/` if keychain results are too crowded, merchant-dominated, or too gadget-driven.

Do not publish both in Phase A unless the first five pages already show clear indexing and outbound-click signal.
